The P-value

In the previous exercise, you have created a multiple regression model with all the variables available. Let's take a look at the results now.

Wow, that's a lot of variables. Do you think all of them are useful for estimating MPG, or are some of them useless? To find out, take a look at the highlighted column. This column is the p-value, which tells us how relevant the variable is to the regression model. The higher the p-value, the more likely that the variable is insignificant. As a default baseline, most studies set the threshold at 0.05. If the p-value is larger than that, then that variable is deemed insignificant for the model, and can be removed. We will disregard the possibility of omitted variable bias, which is outside of the scope of this tutorial.

Ask yourself, Which variables should we keep, and which variables should we exclude?
